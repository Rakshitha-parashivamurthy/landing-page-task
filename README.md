Landing Page – Responsive Web Design

A simple and fully responsive Landing Page built using HTML5 + CSS3, implementing Flexbox, CSS Grid, and media queries to adapt across desktop, tablet, and mobile screens.

 Features

 Clean and semantic HTML structure

 Responsive layout using Flexbox & Grid

 Mobile navigation menu with toggle

 Hero section with heading, description & CTA

 Features section using CSS Grid

 Footer with social links

 Follows accessibility and best practices

 Technologies Used

HTML5 — structure

CSS3 — styling, responsiveness

Flexbox & CSS Grid — layout

Media Queries — mobile responsiveness

VS Code + Live Server — development environment

 How to Run the Project (VS Code)

Download or clone this repository

Open the folder in VS Code

Install Live Server extension (if not installed)

Right–click on index.html → Open with Live Server

The page will open in your browser and auto-refresh on every save

 Responsiveness

The layout adjusts automatically at these breakpoints:

900px → Mobile navigation activated

580px → Single-column layout for hero & features

Test using:

Browser resizing

Chrome DevTools → Device Toolbar
